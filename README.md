# foundation-dogfood

Throwaway sandbox for validating the **Foundation** delivery pipeline end-to-end (Phase 3 dogfood).
Not a real product — it exists to exercise `issue → branch → commit → review → merge` with the agent roster.

## Dev
- Node ≥ 20. **No external dependencies.**
- Test: `npm test` (runs `node --test`).

See [`PROJECT_STRUCTURE.md`](./PROJECT_STRUCTURE.md) for the deployment model and conventions.
